Now to make a list of links! Three types to be exact.

  • Absolute links are the most detailed. They come in three parts. Scheme which starts with http:// or https://, then domain name, and last the path to target web page. Mozilla Developer Network. Absolute links are great for directing users to an offsite webpage.

  • Relative links are from the same scheme and domain. Example would be files from the same folder. You navigate from a file to another file in the same directory basically or deeper(folder inside folder)into the directory. Hi neighbor!. To go up a folder you would use syntax double dot .. before the initial folder location of current page. So if you were on extras.html the directories would look like "a href='../links.html". To go up multiple just keep adding more double dots with foward slashes like '../../../whatever.html'.

  • Root-relative links are webpages that come directly from the "root" of the entire web page itself. This only works if the webpage was the host server, and an exampoe will be, Home Page. This link will not until a host server is created for the root directory. Root relative will start it's directorty with a foward slash.

  • Link targets are a modification of "a" links. They are attributes to be added to the link target itself.Mozilla Developer Network , this link will now open in a new tab or window.
